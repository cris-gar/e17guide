Tangent Bindningar

  2. Programmera Tangenter
  3. Standard Tangent Bindningar

På Enlightenment (E17) språk, syftar Tangent Bindningar på Tangentbords-Genvägar som du kan konfigurera för att underlätta navigering på skrivbordet och en hel rad med funktioner genom att endast använda tangentbordet.

Du kan t.o.m. konfigurera en tangentkombination som startar ett skript eller ett program.

Tangent Bindningar kan konfigureras under Main Menu -> Settings -> Settings Panel -> Input -> Keys, som öppnar Key Bindings Settings dialogrutan:

Det finns två spalter. Den vänstra spalten Key Bindings, listar individuella tangenter och tangentkombinationer. Den högra spalten Action, listar alla tillgängliga händelser som kan triggas. Många tangenter har redan en standardbindning (se tabellen längst ner på denna sida), men du kan ändra bindningen av vilken tangent som helst och lägga till nya kombinationer vid behov.

Utforska inställningar för Tangent Bindningar

För att få en känsla för hur Key Binding Settings (inställningar för Tangent Bindningar) fungerar klicka på några av tangenterna i Key Bindings spalten och lägg märke till vad som händer i Action spalten till höger.

T.ex. till vänster under Single Key, finns en tangent som kallas Menu. Klicka på den i Key Bindings, så visas vad den är konfigurerad att göra, i högerspalten, Show Main Menu. (Meny tangenten är på många tangentbord märkt med en "Meny" symbol, och är oftast placerad mellan AltGr och högra Ctrl tangenterna).

Andra Ensamma Tangenter är listade nedanför Menu. XF86* serien av tangenter finns på många tangentbord för att styra hårdvarufunktioner som t.ex. högtalarvolym, ljusstyrka på skärm o.s.v. klicka på XF86AudioMute, så ser du att den är bunden till Mute Volume.

Under ensamma tangenterna grupperas resten av Tangent Bindningarna i olika kategorier:

  • Shift
  • Ctrl
  • Alt
  • Win (Meta tangenten (oftast med Windows symbolen))
  • Ctrl+Shift
  • Alt+Shift
  • Ctrl+Alt

*OBS! att ovanstående tangenter och kombinationer (undantaget "Win" tangenten) kan inte prgrammeras till att utföra något själva utan måste paras med ytterligare en tangent.

I varje kategori ser du de tangenter som redan har en bindning, och genom att klicka på dem, visas vad de programmerats att göra. T.ex. under Shift kategorin, är kombinationen Shift+F10 den enda Shift tangent- kombinationen som fördefinierats. Klickar du på Shift+F10 i vänstra spalten, visas och väljs Maximize Vertically i högra spalten. Detta maximerar det fokuserade fönstrets lodräta höjd. Observera att det finns många andra möjliga kombinationer som kan programmeras med Shift tangenten inklusive (men inte begränsat till): Shift + A t.o.m. Ö, 1 t.o.m. 0, F1 t.o.m. F9, F11, och F12.

I den högra spalten finns en fullständig lista på tillgångliga kommandon. Vad som finns tillgängligt beror på vilka Moduler som är laddade och är grupperade i kategorier. Listan nedan är ett exempel på vanliga kategorier:

  • Acpi
  • Calendar
  • Desktop
  • Enlightenment
  • Enlightenment Mode
  • Everything Launcher
  • Gadgets
  • Generic: Actions
  • Launch
  • Menu
  • Mixer
  • Pager
  • Screen
  • System
  • Window:Actions
  • Window:List
  • Window:Moving
  • Window:State

Klickar man på någon Action visas inte om det redan finns en tangentkombination redan programmerad till att utföra detta. Du kan kolla i tabellen nedan efter vad som är förprogrammerat i Bodhi Linux installationen.

Programmera Tangenter

Låt oss gå igenom några enkla exempel som visar hur man ändrar existerande bindningar och hur man lägger till nya.

Ändra Ctrl+Alt+S

Som synes i tabellen nedan, har kombinationen Ctrl+Alt+S programmerats att växla mellan Sticky Mode under Window:State kategorin. Du kanske hellre vill använda den kombinationen för att försätta datorn i viloläge (Suspend). Suspend finner du under System kategorin.

I vänstra spalten, klicka på Ctrl+Alt+S. I högra spalten, klicka på Suspend i System kategorin, sen klickar du OK eller Apply. Om du nu trycker på Ctrl+Alt+S, öppnas ett litet fönster som frågar om du vill försätta datorn i viloläge. Klicka Yes för att göra detta, eller No för att avbryta.

Byt Tangent Bindning för Maximera Lodrätt

Låt oss säga att du inte gillar Shift+F10 för att Maximize Vertically (Maximera lodrätt) och helre vill använda Shift+0 för detta. Välj Shift+F10 i Key Bindings spalten, klicka sen Modify Binding (ändra bindning). En liten ruta öppnas där du ombeds trycka in en tangentkombination. Tryck Shift+0, sen OK eller Apply. (När du skriver in tangentkombinationen märker du kanske att detta verkar ändra på vissa andra val i Key Bindings listan. Detta är en liten bug. Om du stänger Key Binding Settings dialogrutan och sen öppnar den igen, ser du att tangent-bindningen blev rätt).

OBS! om du skriver in en Tangent Bindning som redan finns, när du använder Modify Binding, ändrar den inte på bindningen utan hoppar istället till den befintliga Tangent Bindningen i vänstra spalten.

Lägg till Alt+L

Vi säger att du vill programmera Alt+L, (som från början inte är bunden) att starta Leafpad.

I vänstra spalten, klicka på Add Binding (lägg till bindning). En liten ruta öppnas där du ombeds trycka in en tangentkombination. tryck på Alt+L. Detta skapar bindningen i Key Bindings sektionen. Se till att du har Tangent Bindningen du just skapade vald. I den högra spalten klickar du nu på Command i Launch kategorin. Skriv leafpad i Action Params delen längst ner i högra spalten. Klicka OK eller Apply. Tryck nu Alt+L, så ser du att Leafpad öppnas.

Standard Tangent Bindningar

Om du någon gång finner att du skapat en salig röra bland dina Tangent Bindningar och vill återgå till ursprungsläget, öppna bara Key Bindings Settings dialogrutan och klicka på Restore Default Bindings (återställ ursprungliga bindningar) längst ner. Klicka OK eller Apply för att spara.
OBS!: Detta tar bort alla Tangent Bindningar du tidigare skapat.
