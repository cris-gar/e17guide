Bodhi-Guiden till Enlightenment
Inställningar

Enlightenment (E17) Inställningar i Bodhi Linux kommer man åt genom Settings Panel (inställningspanelen) under Main Menu -> Settings -> Settings Panel. Vart och ett av inställnings-dialogrutorna kan även nås direkt under Main Menu -> Settings -> All.

Observera att vissa inställningsmöjligheter endast är tillgängliga när särskillda Moduler har laddats.

Inställning

Beskrivning

Apps (Applikationer)

New ApplicationGUI för att skapa .desktop filer som kan läggas till i Huvudmenyn.
Favorite ApplicationsLägg till/ta bort inslag i Favoriter menyn. Låter dig även byta ordningsföljd.
IBar Applications Lägg till/ta bort applikationer i IBar Gadgeten. Låter dig även byta ordningsföljd.
Restart Applications Välj applikationer som skall köras automatiskt efter omstart av E17.
Startup Applications Välj applikationer som skall köras automatiskt vid uppstart.

Screen (Skärm)

Virtual DesktopsStäll in antal och uppläg av virtuella skrivbord, även animeringen när man skiftar mellan dem.
Screen Resolution Ställ in skärmupplösning.
Screen Lock Ställ in skärm lås tiden, inoggningsfönstrets placering, bakgrundsbild när låst mm.
Screen Saver Aktivera/avaktivera X skärmsläckare, aktiverings-tid mm.
Power Management Aktivera/avaktivera strömhantering, ställ in väntläge, viloläge, avstängnings-tid mm.

Windows (Fönster)

Window Display Ställ in beteende för fönstervisning - flytta, ändra storlek, nya fönster, skuggning mm.
Window Focus Ändra på hur muspekaren ändrar fokus av fönster. Avancerade inställningar tillgängligt.
Window Geometry Ändrar fönstrens motstånd till andra element på skärmen och parametrar vid ändring av storlek och läge via tangentbordet.
Window Remembers Använd automatiskt senaste inställningarna för interna dialogrutor eller filhanterarfönster nästa gång de öppnas. Kan minnas alla eller delar av fönsterinställningar inklusive storlek, läge, lås, mm. Denna dialogruta är främst avsedd för att radera "window remembers" (fönsterminne). Window remembers ställs in med fönster-menyn.
Window Stacking Inställningar för att höja & sänka fönster (djupled) och även för att tillåta fönster ovanför fullskärm.
Window Maximize Policy Ställ in beteende när fönster maximeras. Valen innefattar Fullskärm, Smart Expansion, Fyll tillgängligt utrymme, och kan även bestämma riktning för fönsterexpansion.
Client List Menu Ställer in hur fönstren listas i Klient Lista-menyn. Åtkomst antingen genom Mus 3/mittknapp-klick eller Main Menu -> Windows.
Window List Ställ in beteende för Fönster-Listan. Vanligtvis åberopad genom Alt+Tab

Menus (Menyer)

Menu SettingsEnlightenment Huvudmeny-inställningar. Ställ in vilken applikationsinformation som visas och skroll-beteende på menyn.

Language (Språk)

Language Settings Ändra lokaliseringinställningar för skrivbordsmiljön.
Input Method Settings Ändra inställningar för diverse inmatningsmetoder vilket medger inmatning av symboler som inte förekommer på den vanliga inmatnings anordningen t.ex. mata in kinesiska tecken med ett svenskt tangentbord.

Files (Filer)

File Manager Ändra inställningar för Enlightenment File Manager (EFM).
File Icons ställ in ikonerna som används av EFM för olika filtyper.
Places Ändra inställningar för Places modulen. Ger tillgång till filhanteraren via Huvudmenyn och användandet av någon annan filhanterare (t.ex. PCManFM).

Advanced (Avancerat)

Performance Fintrimma E17s prestanda. Medger justering av bildhastighet, font cache, bild cache, edje cache, o.s.v.
Engine Tillåter användandet av ARGB fönster istället för formade fönster.
Search Directories Visa/ställ in sökvägar för mappar som används av E17 för att hitta särskillda filer T.ex. Bilder, Fonter, Ikoner, o.s.v.
Battery Meter Styr hur batteriet övervakas. Ställ in pollnings tid, kriteria för lågt batteri varning, o.s.v.
Configuration Panel Aktivera/avaktivera vilka E17 inställnings-dialogrutor som ska vara tillgängliga från Main Menu -> Settings -> All.
Environment Variables Ställ in anpassade E17 variabler för system/skrivbords-miljö.

Look (Utseende)

Wallpaper Byt bakgrundsbild för alla skrivbord eller nuvarande skrivbord.
Wallpaper 2 En pråligare dialogruta för att byta bakgrundsbild för alla skrivbord eller nuvarande skrivbord.
ThemeByt Enlightenment tema. Avancerade valet låter en tilldela teman till individuella element.
Colors Tilldela in anpassade färger på individuella Enlightenment element. Färger är beroende av temats möjligheter.
Fonts Tilldela anpassade font inställningar i Enlightenment. Avancerade valet låter en applicera anpassningen på individuella element.
Borders Ställ in standard stil på fönsterramar.
Icon Theme Ställ in ikontema att användas i Enlightenment.
Mouse Cursor Välj mellan X eller Enlightenment stil på muspekare, ställ även in storleken.
Transitions Ställ in typ av övergång som används vid vissa händelser t.ex. vid byte av bakgrundsbild.
Scaling Skala övergripande fontinställninen i Enlightenment.
Startup välj startbildsanimering som används när systemet bootas.
Composite Scale Windows Inställningar för animerings när scale windows används. Scale Windows är en alternativ metod för att förflytta sig mellan öppna fönster.
CompositeStäll in Composite Modul effekter och fintrimma Composite prestanda.

Input (Inmatning)

KeysInställningar för Tangent-Bindningar (Tangentbords-genvägar).
Edge Bindings Liknande. Ställ in händelser som sker när muspekaren träffar olika delar av skärmkanten.
Mouse Buttons Ställ in bindningar till musknappar. Musknappar kan kombineras med tangenter för ytterligare bindningar.
Mouse Välj orientering på musknappar och accelerationshastighet.
Hardware Switches Ställ in specialiserade hårdvarubindningar (ACPI) t.ex. tryck på På/Av knappen, stäng skärmlocket på laptopen mm.
Touch Inställningar för pekplatta.

Settings (Inställningar)

Dialogs Ställ in val för dialogrutor, om de skall bete sig som vanliga fönster och/eller om tillgängligt huruvida de automatiskt skall öppnas i standard eller avancerat läge.
ProfilesSkapa, radera, och växla mellan olika Profiler.

Launcher (Startare)

Everything ConfigurationStäll in Run Everything programmet, hur val sorteras, visa typer, storlek, uppställning, och aktiva insticksprogram.
Everything Plugin: Applications Ställ in terminal och GUI commandon för att starta applikationer.
Everything Plugin: Files ställ om den skall söka igenom och/eller visa de senaste filerna eller cachen. Låter en äver rensa cache.
Everything Plugin: Plugins Allmäna inställningar för insticksprogram i Run Everything.
Everything Plugin: Start Ställ in ordningen på Everything insticksprogram.

Extensions (Tillbyggnader)

ModulesLadda/Ladda-Ur Enlightenment moduler.
ShelvesLägg till/Ta bort Hyllor och ändra inställningar för individuella Hyllor.
Mixer Ställ in vilken mixer som används för ljud, och öppna inställningar för mixer-kanal.
Pager Inställningar för Pager Gadgets.
Engage Lägg till, ta bort, ändra i förekomster av Engage Gadgeten.
Connection Manager Visar tillgängliga nätverk och ställ in åtkomst.
Gadgets Lägg till tillgängliga Gadgets till skrivbordet.

System

**Dessa inställningar kräver root behörighet
NTFS Configuration Tool Aktivera/avaktivera skriv stöd till NTFS enheter.
Software Sources Välj vilka repositories att avsöka för tilgängliga mjukvarupaket och uppdateringar.
LXDM Autologin Ställer in användaren för automatisk inloggning för att kringå initiala inloggningsskärmen.
Elementary Configuration Inställningar för widgets (liknande Gadgets) som använder sig av Elementary widget setet.
Synaptic Package Manager Installera, avinstallera, ladda ner eller uppdatera mjukvarupaket från tillgängliga repositories.
LXAppearance Anpassa utseende och känsla på LXDE applikationer t.ex. PCManFM.
