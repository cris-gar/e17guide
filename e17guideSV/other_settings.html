Bodhi-Guiden till Enlightenment
Övriga Inställningar

Denna artikel kommer att ta upp en del övriga inställningsmöjligheter som inte direkt hör till eller berörs av Enlightenment (E17). Många applikationer kan inte konfigureras av E17. Denna del är med för de användare som ev. vill ändra på vissa aspekter av dessa applikationer.

Grundläggande Tema-anpassning

E17 har lite inbyggd funktionalitet för att styra utseendet på GTK/QT applikationer. För att komma åt detta gå till Main Menu -> Settings -> All -> Look -> Applications. Du kommer att få en ruta som listar alla GTK/QT teman du har installerade.

Välj helt enkelt det GTK/QT tema du vill använda, klicka Apply. När du hittat ett tema du är nöjd med, klicka Close.

E17 + GTK/QT Integrering

Vissa E17 teman har ett definierat GTK/QT tema som matchar. Detta medger att GTK/QT temat byts automatiskt samtidigt som E17 temat byts (se Themean för mere information). För att aktivera denna funktion, öppna Main Menu -> Settings -> All -> Look -> Applications och bocka i rutorna för Match E17 theme if possible och Enable Settings Daemon.

Notera att det finns många E17 teman som inte har något fördefinierat GTK/QT tema, vilket innebär att byte av E17 temat inte påverkar GTK/QT temat.

Det är möjligt att till fullo styra alla aspekter av ett GTK/QT tema genom att installera applikationen LXAppearance ...

LXAppearance

LXAppearance finns inte med i grundinstallationen av Bodhi Linux. För att installera det, kör kommandot:

sudo apt-get install lxappearance

Tillgång till det får du genom Main Menu -> Applications -> Preferences -> LXAppearance.

Varje flik längs ovansidan låter dig styra olika aspekter.

Widget: Denna flik låter dig välja mellan en mängd fördefinierade GTK/QT teman.
GTK/QT teman som du installerat (se the Bodhi Art Site - internet anslutning krävs) och de teman som installerats som standard kommer att synas i listan på vänster sida. Ytan på höger sida visar en förhandsbild av varje tema. Nedanför detta kan Font väljas - detta påverkar endast fonten i aktiva GTK/QT applicationer och ändrar inte övriga systemets fonter.

Color (Färg): Denna flik låter dig skapa ditt eget färgschema för GTK/QT applikationer. Use customized color scheme rutan måste bockas i for att detta alternativ skall vara tillgänglig.

Icon theme: Ikon-set kan förhandsgranskas och väljas i denna flik. Se Bodhi Art Site (online länk) för fler ikoner!

Mouse Cursor: Om du installerat andra muspekare, kan de väljas här. Detta påverkar endast muspekaren medan den befinner sig över en aktiv GTK/QT applikation.

Font: I denna flik ges du möjlighet att göra inställningar som påverkar font-glättning (gör fonten mindre kantiga/taggiga).

Other: Dessa valmöjligheter är specifikt för fönsterhanterare som byggts med GTK/QT och har därför ingen påverkan på ett E17 skrivbord.
