Inledning

Menyn är en allmänt förekommande del av ett modernt Grafiskt Operativsystem och det åberopas vanligtvis genom att klicka på en knapp eller ikon på en panel eller aktivitetsfält. T.ex. den ökända windows Start knappen.

Enlightenment (E17) tillhandahåller detta vanliga användargränssnitt genom att använda sig av en s.k. Start Gadget som kan placeras var som helst på skrivbordet - vanligtvis placeras denna i en Shelf (hylla). De flesta Profiler som installeras i Bodhi Linux innehåller en ”Start Knapp” och i de flesta Teman ser den ut som ett Bodhi Linux blad.

Det är emellertid fullt möjligt att ta bort Start knappen. Att göra något sådant kan ju verka märkligt om man är obekant med E17. Ingen fara, detta tar inte bort menyn utan endast knappen. E17's huvudmeny kan visas genom att klicka på vilken tom yta på skrivbordet som helst. Observera att E17 har två menyer. Menyn du får fram genom att klicka på Start knappen eller vänsterklicka på en tom yta på Bodhi Linux skrivbordet är den så kallade Huvudmenyn, medan menyn som visas när man högerklickar på nogon tom yta på skrivbordet är Favoriter Menyn. På grundinstallationen av Bodhi Linux, är Favoriter Menyn tom, men som vi kommer att märka så är det enkelt att lägga till dina egna favorit-applikationer i denna meny.

Utöver åtkomst till E17's Huvudmeny från antingen Start knapp eller vänsterklick på skrivbordet finns det i Bodhi Linux även en förinställd key-binding (tangent-bindning) som öppnar Huvudmenyn från tangentbordet. Det är Meny-Tangenten! På de flesta tangentbord finner du denna tangent till höger om mellanslag, mellan AltGr and Ctrl tangenterna och ser ut ungefär som på bilden till höger. Naturligtvis kan denna bindning ändras eller tas bort helt.

Vart försvann det?

En liten egenhet med E17's Meny som kan förvirra nya användare av Bodhi Linux. Alltefter du skrollar ner för listan i Huvudmenyn kommer undermenyerna alltid att öppnas till höger om föregående meny. Vanligtvis skapar detta inga bekymmer, men vad händer om du öppnar huvudmenyn nära skärmens högerkant? I det här fallet kan undermenyerna hamna delvis eller helt utanför skärmen. Det är till synes något besynnerligt med E17's implimentering av menyer, den tanken är dock något förhastad. I det här fallet behöver du bara flytta muspekaren åt höger så kommer hela menystrukturen att förflytta sig åt vänster och visa de tidigare dolda underliggande menyvalen. Som programmerare gärna säger, ”Det är ingen bug, det är ett särdrag.” Ännu ett exempel på hur elegant och vackert Enlightenment är.

Menystruktur

Detta avsnitt ger en kort beskrivning av Bodhi Linux Huvudmeny's uppbyggnad, fast om sanningen skall fram så är det bästa sättet att bekanta sig med den att öppna den och utforska sjäv. I vilket fall som helst, Huvudmenyn på en grundinstallation av Bodhi Linux listar, uppifrån och ner:

  • Applikationer
    Innehåller de flesta GUI applikationerna installerade på ditt system, ordnade i undermenyer baserade på kategori. Kommandorad-applikationer visas inte i Applikations-menyn. De kan emellertid läggas till.

  • Platser
    Grundinstallationen av Bodhi Linux, Platser låter dig öppna Hemmakatalogen, Sopkorgen, (root) filsystemet, "mountade" filsystem och standard-Filhanterarens bokmärken. Platser är igentligen en Module och kan därför "laddas-ur" systemet och försvinner då ur menyn. Du kan även ändra grundinställningarna för Platser Modulen. För att göra detta från Huvudmenyn gå till Settings → All → Files → Places. Detta öppnar en Places Settings dialogruta där du kan ändra Platser till att visa eller inte visa följande: Home, Desktop, Trash, File system, Temp, och Favorites. Du kan även välja att inte alls visa Platser i Huvudmenyn.

  • Run everything
    Startar Run Everything Applikationen.

  • Desktop
    Ett enkelt sätt att växla mellan virtuella skrivbord Och snabb tillgång till Virtuella Skrivbord Inställningar och Shelves Inställningar. Du har även möjlighet att visa eller dölja alla öppna fönster på det aktuella skrivbordet.

  • Windows
    Ger snabbt tillgång till varje öppet fönster, möjlighet att rensa upp bland fönster och möjlighet att hitta borttappade fönster.

  • Bodhi Linux
    Här finner du Bodhi Linux's utbyggnad på E17's Huvudmeny. Det finns fyra sektioner:
    • About
      Öppnar Om Bodhi Linux rutan.
    • Quick Start
      Öppnar en lokan version av Snabbstart websidan i standard-webläsaren (engelsk version).
    • Add Software
      Öppnar Bodhi Linux AppCenter websidan i standard-webläsaren. Härifrån kan du enkelt och smidigt installera användbara applikationer mm.
    • Artwork
      Öppnar Bodhi Konst websidan där du kan ladda hem extra page where you can get some extra "eye candy" (godis för ögonen) för ditt skrivbord. De flesta som bidrar med konst på denna sida är själva Bodhi Linux användare snarare än professionella grafiker/konstnärer.

  • Enlightenment
    • About
      Öppnar Om Enlightenment rutan.
    • Theme
      Öppnar Om Thema rutan som visar uppgifter om det nuvarande E17 temat.
    • Restart
      Väljer du detta, startas E17 om. Kan ibland vara nödvändigt om inställningar inte verkar vilja ändras, eller om man efter en applikationsinstallation inte finner denna i menyn, eller om mot förmodan E17 inte verkar fungera som det ska.
    • Quit
      Stänger ner E17.

  • Settings
    Öppnar E17's Inställningar-Panel. Paradiset för de som vill anpassa allt i sitt system!

  • System
    Här finns de välbekanta Shutdown (stäng av), Reboot (starta om), Suspend (viloläge), Lock (lås), mm. funktionerna.

Värt att notera är att om man laddar vissa Moduler eller ändrar i vissa E17 Inställningar kan vissa menyobjekt tillkomma eller försvinna. Utöver detta, lägger man till applikationer i Favoriter Menyn tillkommer Favorit-Applikationer överst på Huvudmeny listan, Vilket leder oss in på nästa ämne.

Favorit-Applikationer

Favoriter Menyn innehåller en lista på applikationer till vilka användaren vill ha snabb tillgång. Denna lista över favorit-applikationer återfinns antingen i Bodhi Linux's Huvudmeny eller genom högerklick på en tom del av skrvbordsytan. På grundinstallationen av Bodhi Linux, syns inte Favoriter Menyn i Bodhi Linux's Huvudmeny. Högerklick för att få tillgång till Favoriter Menyn fungerar inte heller. Detta kommer sig av att inga applikationer lagts i favoritlistan som standard. Det är del av Bodhi Linux filosofi att inte försöka förutspå vad användarna vill ha, behåll det minimalt och låt användarna själva få välja.

Att lägga till Lxterminal i Favoriter

Det finns två metoder för att lägga till en applikation i Favoriter Menyn. Första metoden; öppna applikationen du vill lägga till, låt oss säga, Lxterminal. Vänsterklicka applikationens ikon i övre vänstra hörnet, välj applikationsnamnet och klicka sedan Add to Favorites Menu (lägg till i Favoriter Menyn) som bilden till höger visar. Lxterminal visas nu i din Favoriter Meny. Otroligt enkelt och verkligen elegant!

Andra metoden är inte mycket svårare. Från Bodhi Linux's Huvudmeny, välj Settings → All → Apps → Favorite Applications. Detta öppnar E17's Favorit-Applicationer ruta, under Applications skrolla ner i listan till du hittar applikationen du vill lägga till, klicka på den för att markera den, sen klickar du på Add knappen. En grön prick syns nu bredvid applikationsnamnet som för PCManFM på bilden till vänster. (OBS! Indikatorn kan vara något annat än en grön prick i vissa Profiler) Klicka Apply och sen Ok. Applikationen läggs nu till i din Favoriter lista och syns nu i menyn.

För att avlägsna en applikation ifrån Favoriter Menyn, öppna Favorit-Applicationer rutan som beskrivs ovan. Skrolla genom applikationslistan. Ha i åtanke att applikationer i Favoriter Menyn har en grön prick (eller annan indikator) vid sitt namn. Hitta applikationen du vill avlägsna, klicka på den för att markera den, sen klickar du på Remove knappen. Gröna pricken (eller annan indikator) vid applikationsnamnet försvinner. Klicka Apply och sen Ok. Applikationen tas nu bort från din Favoriter lista.

Om du är en flitig användare av Favoriter Menyn och ofta använder en mängd applikationer kan du finna att din Favoriter Meny blir rätt lång och att du kanske vill byta ordning på applikationerna, kanske lägga de mest använda applikationerna överst i listan. Självklart låter E17 dig byta plats på applikationerna i listan. Åter igen, öppna E17's Favorit-Applicationer, denna gång klickar du på Order fliken, se bilden till höger. Hitta applikationen du vill flytta, klicka för att markera, sen klickar du antingen på Up eller Down knappen för att placera den där du vill ha den i menyn. Om så önskas kan du göra detta med en eller fler andra applikationer. När du väl är nöjd med ordningen, klicka Apply och sen Ok.


Anpassa Bodhi Linux Menyn

E17 har ett ganska unikt tillvägagångssätt när det kommer till redigering, tillägg eller avlägsnande av menyobjekt. Istället för att ha en separat applikation som låter användaren ändra i menyn, är funktionen att grafiskt redigera menyobjekt inbyggt i själva E17. För att se hur, låt oss titta på hur man lägger till en applikation i Huvudmenyn.

Lägg till en Applikation

Ibland händer det att man installerar en applikation och så dyker den inte upp i menyn. Detta beror förmodlien på hur applikationen packeterades. För våra ändamål i denna guide kommer vi att använda oss av två applikationer som redan är installerade på Bodhi Linux. Dessa applikationer är Archive Manager (Arkivhanteraren) och xterm. På en grundinstallation av Bodhi Linux, ligger ingen av dessa applikationer i Huvudmenyn. De två procedurer som beskrivs nedan för att lägga till dessa applikationer skall vara tillämpbara på viken applikation du än installerar som av någon anledning inte lägger till en listning i Huvudmenyn.

Först tar vi oss an Arkivhanteraren. Nyckeln här är att starta applikationen. Det finns tre eller fler sätt att göra detta på. Ett sätt är att använda E17's Run Everything, ett annat är att starta applikationen från ett terminalfönster och det tredje att hitta applikationsfilen och starta genom att klicka på den.

För att visa detta med Run Everything applicationen, öppnar vi först Run Everything. Den hittar du i Huvudmenyn eller startar från kortkommandot, Alt-Esc, Ctrl-Alt-Mellanslag eller Meta(Windows)-Mellanslag. När den öppnats, börja skriva Archive Manager och så fort du ser att den visas dubbeklicka på ikonen för att starta den.

När Arkivhanteraren väl öppnat, högerklicka i arkivhanterarens tittellist eller på dess ikon överst till vänster på skärmen och välj File-roller → Edit Icon. Ett Desktop Entry Editor (Skrivbords-tilläggs-redigerings) fönster kommer att öppnas, se nedan. Klicka på Options, bocka sedan i Show In Menus, klicka Apply och Close. Du finner nu Arkivhanteraren i menyn under Accessories (Tillbehör).

För den andra metoden låt oss lägga till xterm i menyn. Denna metod är igentligen bara ett annat sätt att öppna Desktop Entry Editor fönstret på. I Huvudmenyn välj Settings → All → Apps → Create Launcher. I Desktop Entry Editor rutan som öppnas fyll i följande poster:

  • Basic Fliken
    Name: XTerm
    Comment: Använd kommandorad
    Application: xterm

  • The Icon Tab
    Icon: terminal

  • General Fliken
    Generic name: Terminal Categories: GTK;Utility;TerminalEmulator;

  • Options Fliken
    Bocka i "Show in Menus"

När du är klar, Klicka Apply och Close. Nu finns xterm i menyn under Accessories.

Ta bort en Applikation

Nu när du har sett hur man använder Desktop Entry Editor verktyget borde det vara uppenbart hur man gör för att ta bort en applikation ur menyn. Starta applikationen du vill avlägsna, vänsterklicka dess ikon uppe till vänster på skrivbordet och välj Applikationsnamn → Edit Icon. En Desktop Entry Editor ruta kommer att öppnas. Klicka på Options och avbocka Show In Menus, sen klickar du Apply och Close. Applikationsnamn i detta fall är naturligtvis namnet på applikationen du vill ta bort och även det första menyobjektet i "vänsterklick på ikonen" menyn.

Enlightenment's Menyinställningar

Det finns ytterligare ett verktyg att beakta för att ändra i Bodhi Linux's Meny gömd i E17's Settings (inställningar). Detta är Menu Settings (menyinställningar), dialogrutan som visas här till vänster. För att öppna denna ruta, öppnar du Huvudmenyn och väljer sen Settings → All → Menus → Menu Settings.

Menu Settings fönstret har fyra flikar: Menus, Applications, Autoscroll, Miscellaneous, (Menyer, Applikationer, Autoskroll, Diverse). Den öppnas med Menus fliken förvald. Fliken är indelad i tre undersektioner:

  • Main Menu
    I eller ur-bockning av någon post här, visar eller döljer den valda undersektionen i Huvudmenyn.

  • Applications Display
    I eller ur-bockning av någon post här, ändrar sättet på vilket en applikation visas i E17's menyer. du kan välja att visa (från exemplet ovan) en applikation's specifika namn (lxterm), dess generella namn (Terminal), och/eller applikationens kommentar (Använd Kommandorad). Dessa poster är samma som du finner i Desktop Entry Editor och kan naturlightvis ändras på där om så skulle behövas.

  • Gadgets
    Denna funktion är utfasad, i eller ur-bockad, gör ingen skillnad.

OBS! För att försäkra sig om att de ändringar som gjorts i Menu fliken eller någon annan flik i Menu Settings fönstret utförs, se till att alltid klicka Apply innan du går vidare, annars utförs/sparas inte ändringarna.

Att klicka på Applications fliken i Menu Setting fönstret, ger dig möjlighet att välja vilken menykälla systemet använder för att visa Applications sub-menu (Applikations-undermenyer). I en standardinstallation av Bodhi Linux är valen Enlightenment och Lxde (Applicationer). Det är ingen skillnad på Application sub-menu oavsett viket du väljer.

De resterande två flikarna Autoscroll och Miscellaneous låter dig justera diverse aspekter av hur E17's menyer beter sig. Det bästa sättet att lära sig vad dessa olika inställningar gör är att prova och leka runt lite med dom.

Ytterligare Information

Vissa inslag i undermenyerna är tillräckligt inveklade för att motivera egna artiklar som beskriver deras användande:

  • Enlightenment Settings: En djupdykning i alla E17's inställningar.
  • Everything Modulen: En grundläggande genomgång av hur man använder Run Everything Modulen.
  • LXAppearance: Denna applikation styr utseendet av GTK applikationer.
